# Markdown Reader v2.1.0

新增命令面板、右键在访达中打开、本地 Markdown 链接点击打开，官网和 README 多语言支持。

## ✨ 新增

### 🔍 命令面板
- 按 `Cmd+P` 打开命令面板，快速搜索并打开目录树中的 Markdown 文件
- 支持实时过滤、模糊匹配文件名和相对路径
- 键盘导航：↑↓ 选择、Enter 打开、Esc 关闭
- 半透明遮罩 + 标题栏下方居中浮层设计
- 菜单栏新增「命令面板…」菜单项

### 📁 右键「在访达中打开」
- 文件树中目录和文件的右键菜单新增「在访达中打开」选项
- 点击后在 Finder 中显示并选中对应文件/目录

### 🔗 本地 Markdown 链接点击打开
- 渲染视图中点击本地 Markdown 文件链接时，自动在应用内打开目标文件
- 目录模式下，目标文件在根目录内时仅切换文件树选中项
- 目标文件在根目录外时，自动切换为单文件模式打开

### 🌐 官网多语言支持
- GitHub Pages 官网新增英文、日文、繁体中文三个语言版本

### 📖 README 多语言版本
- 新增英文、日文、繁体中文 README 文件

## 🐛 修复

- 本地 Markdown 链接导航：通过文档加载流程打开链接文件，确保状态一致性

## ⌨️ 新增快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd+P` | 打开命令面板 |

## 🖥️ 系统要求

- macOS 26 (Tahoe) 或更高版本
- Apple Silicon 原生支持
