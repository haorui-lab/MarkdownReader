# Markdown Reader — 需求文档

## 1. 产品概述

Markdown Reader 是一款 macOS 原生应用，用于浏览和阅读 Markdown 文件。采用 SwiftUI + cmark-gfm + WebPage（macOS 26 原生 SwiftUI WebView）构建，提供三栏布局：左侧目录树导航 + 中间 Markdown 渲染显示 + 右侧大纲导航，遵循 macOS 标准设计风格。灵感源自 Buddy macOS 设计。

## 2. 目标用户

- 需要本地浏览 Markdown 文档的开发者和技术人员
- 需要阅读本地知识库/笔记的用户
- 需要快速预览 Markdown 文件的用户

## 5. 约束

- 技术栈：SwiftUI + cmark-gfm + WebPage（macOS 26 SwiftUI 原生 WebView，v2.x 从 Textual 迁移）
- 仅支持 macOS 平台
- 只读浏览，不支持编辑
- 不做同步/云端功能
- Swift 6.0 + macOS 26 使用 `@Observable` 宏
- Word 式多窗口：同一 Markdown 文件不允许多窗口同时存在，再次打开激活已有窗口（v2.2.0）
- 仅支持 Apple Silicon (arm64)

## 6. 不做的范围（明确排除）

- Markdown 编辑功能
- 云端同步
- 同一文件多窗口编辑（多窗口下同一文件只允许一个所有者窗口）
- 标签页/分屏平铺
- 完整窗口会话恢复
- 插件系统
- 原文视图行号显示（P2）
