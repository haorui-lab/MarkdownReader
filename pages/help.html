快速入门

1

打开文件或文件夹

按 ⌘O 或点击工具栏打开按钮，选择 .md 文件或文件夹。也可以直接将文件或文件夹拖拽到窗口中。

2

阅读和导航

左侧文件树浏览目录结构，右侧渲染视图阅读内容。点击工具栏大纲按钮或按 ⌘\ 切换侧边栏。

3

编辑与保存

切换到原文模式（⇧⌘R）直接编辑，按 ⌘S 保存。切换文件时未保存内容会自动保留。

功能详解

渲染视图

默认的渲染模式使用 cmark-gfm 引擎解析 Markdown，通过 WKWebView 渲染为富文本。支持完整 GFM 扩展语法，包括表格、任务列表、删除线、自动链接等。

原文编辑

按 ⇧⌘R 切换到原文模式，直接编辑 Markdown 源码。支持语法高亮（Prism.js 引擎，30+ 语言）。按 ⌘S 保存。切换到其他文件时，未保存的内容会自动保留，返回时恢复。

文件树

打开文件夹后，左侧显示递归文件树。支持键盘导航（方向键选择，回车打开）。右键菜单提供新建文件/子目录、重命名、移动、删除等操作。可在设置中开启显示隐藏文件和非 Markdown 文件。

大纲导航

自动提取当前文档的标题层级结构。点击大纲中的标题可快速跳转到对应位置，阅读长文档更高效。

查找与替换

按 ⌘F 打开查找栏。支持区分大小写、全词匹配、正则表达式。⌥⌘F 打开查找与替换模式。

视图缩放

使用 ⌘+ / ⌘- 缩放内容，范围 0.3x–3.0x。按 ⌘0 恢复实际大小。缩放级别在重新加载时自动恢复。

主题与外观

主题模式

在设置 → 外观中，可选择浅色、深色或跟随系统三种模式。应用会立即切换外观。

配色方案

提供 33 套预设配色方案（20 深色 + 13 浅色），包括 Dracula、Catppuccin、Nord、Tokyo Night、Markdown Preview Enhanced 风格等。每种模式有独立的配色，切换模式时自动应用对应方案。

自定义颜色

在设置 → 外观 → 自定义颜色中，可单独调整背景色、文字色、强调色、成功色、危险色。自定义颜色会覆盖当前方案中的对应项。

对比度调节

提供低/高两档对比度选项，调整背景与前景层之间的视觉对比。

图表与公式

Mermaid 图表

在代码块中使用 mermaid 语言标识，即可渲染流程图、时序图、甘特图等。本地渲染，无需网络。

```mermaid
graph LR
    A[开始] --> B{判断}
    B -->|是| C[执行]
    B -->|否| D[结束]
```

数学公式 (KaTeX)

使用 $...$ 包裹行内公式，$$...$$ 包裹块级公式。

行内公式：$E = mc^2$

块级公式：
$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$

PlantUML 图表

在代码块中使用 plantuml 语言标识，自动渲染为 SVG 图表。需要网络连接 PlantUML 服务器。

```plantuml
@startuml
Alice -> Bob: Hello
Bob --> Alice: Hi
@enduml
```

命令行工具 mdr

安装

打开 设置 → 通用 → 命令行工具，点击「安装」按钮。安装完成后，mdr 命令即可在终端中使用。

使用

# 打开单个文件
mdr README.md

# 打开多个文件
mdr doc1.md doc2.md

# 打开文件夹
mdr ./docs

Quick Look 预览

在 Finder 中选中 .md 文件，按空格键即可预览渲染效果，无需打开应用。此功能需在 设置 → 通用 → Quick Look 预览 中启用。

导出 PDF

按 ⌥⌘E 或使用菜单 文件 → 导出 PDF，将当前渲染内容导出为 PDF 文件。导出的 PDF 保留当前主题的样式和配色。

常见问题

支持哪些文件格式？

支持 .md、.markdown、.mdown、.mkd 等 Markdown 格式文件。

系统要求是什么？

macOS 26 (Tahoe) 或更高版本。

PlantUML 图表无法渲染？

PlantUML 渲染需要网络连接，请确保可以访问 PlantUML 服务器。如果处于离线环境，Mermaid 图表仍可正常使用（本地渲染）。

如何设置 Markdown Reader 为默认打开程序？

打开 设置 → 通用 → 默认 Markdown 打开程序，点击「设为默认」即可。也可以右键 .md 文件 → 显示简介 → 打开方式 → 选择 Markdown Reader → 点击「全部更改」。
