快速入门

打开文件，开始阅读，就这么简单

1. 打开文件或文件夹

按 ⌘O 或点击工具栏打开按钮，选择 .md 文件或文件夹。也可以直接将文件或文件夹拖拽到窗口中。

2. 阅读和导航

左侧文件树浏览目录结构，右侧渲染视图阅读内容。点击工具栏大纲按钮或按 ⌘\ 切换侧边栏。

3. 编辑与保存

切换到原文模式（⇧⌘R）直接编辑，按 ⌘S 保存。切换文件时未保存内容会自动保留。

功能详解

每个功能都为阅读体验而生

渲染视图

cmark-gfm 引擎解析 Markdown，WKWebView 渲染为富文本。支持完整 GFM 扩展语法，包括表格、任务列表、删除线、自动链接等。

原文编辑

按 ⇧⌘R 直接编辑 Markdown 源码，Prism.js 语法高亮支持 30+ 语言。⌘S 保存，切换文件自动保留未保存内容。

文件树

递归浏览文件夹，键盘导航（方向键选择，回车打开）。右键菜单提供新建、重命名、移动、删除等操作。

大纲导航

自动提取标题层级结构，点击快速跳转，阅读长文档更高效。

查找与替换

按 ⌘F 打开查找栏。支持区分大小写、全词匹配、正则表达式。⌥⌘F 打开查找与替换。

视图缩放

使用 ⌘+ / ⌘- 缩放内容，范围 0.3x–3.0x。按 ⌘0 恢复实际大小，缩放级别自动恢复。

主题与外观

33 套主题，打造你的阅读空间

主题模式

在设置 → 外观中，可选择浅色、深色或跟随系统三种模式，应用立即切换外观。

配色方案

33 套预设配色（20 深色 + 13 浅色），Dracula、Catppuccin、Nord、Tokyo Night……每种模式有独立配色。

自定义颜色

在设置 → 外观 → 自定义颜色中，可单独调整背景色、文字色、强调色、成功色、危险色。

对比度调节

提供低/高两档对比度选项，调整背景与前景层之间的视觉对比。

图表与公式

技术文档的完美渲染

Mermaid 图表

在代码块中使用 mermaid 语言标识，即可渲染流程图、时序图、甘特图等。本地渲染，无需网络。

数学公式 (KaTeX)

使用 $...$ 包裹行内公式，$$...$$ 包裹块级公式。LaTeX 语法完美呈现。

PlantUML 图表

在代码块中使用 plantuml 语言标识，自动渲染为 SVG 图表。需要网络连接。

命令行工具 mdr

终端工作流无缝衔接

安装

打开 设置 → 通用 → 命令行工具，点击「安装」按钮。安装完成后，mdr 命令即可在终端中使用。

使用

终端输入 mdr README.md 即可打开文件，支持同时打开多个文件和文件夹。

更多功能

还有这些贴心功能等你发现

Quick Look 预览

在 Finder 中选中 .md 文件按空格，即可预览渲染效果。需在 设置 → 通用 → Quick Look 预览 中启用。

导出 PDF

按 ⌥⌘E 或使用菜单 文件 → 导出 PDF，将渲染内容导出为 PDF 文件，保留当前主题样式。

常见问题

遇到问题？先看看这里

支持哪些文件格式？

支持 .md、.markdown、.mdown、.mkd 等 Markdown 格式文件。

系统要求是什么？

macOS 26 (Tahoe) 或更高版本。

PlantUML 图表无法渲染？

PlantUML 渲染需要网络连接，请确保可以访问 PlantUML 服务器。如果处于离线环境，Mermaid 图表仍可正常使用（本地渲染）。

如何设置 Markdown Reader 为默认打开程序？

打开 设置 → 通用 → 默认 Markdown 打开程序，点击「设为默认」即可。也可以右键 .md 文件 → 显示简介 → 打开方式 → 选择 Markdown Reader → 点击「全部更改」。
