# Markdown Reader v1.0.1

Bug 修复与功能增强版本。

## ✨ 新增

### 📂 文件外部修改检测
- 实时监控当前文件变更，未修改文件自动静默刷新
- 已修改文件在标题栏显示刷新按钮，提醒用户文件已被外部应用修改
- 侧边栏右键菜单新增「重新加载」选项
- 有未保存修改时弹出确认对话框，支持「以后不再提醒」

### 🔧 双击文件启动优化
- 通过 Finder 双击文件启动应用时，优先打开用户点击的文件，不再错误恢复上次位置
- 新增 AppDelegate 处理冷/热启动文件打开事件，防止重复打开

### 🛠️ 其他
- 禁用 macOS 窗口标签页功能，隐藏「显示标签页栏」菜单项
- 新增 7 个本地化键值（简中/繁中/英文）

## 🖥️ 系统要求

- macOS 15.0 (Sequoia) 或更高版本
- Apple Silicon / Intel 均支持
